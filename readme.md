# Como Jogar
Utilize as `teclas`:

    "w" ou "seta_cima" = ir para cima; 

    "s" ou "seta_baixo" = ir para baixo;

    "d" ou "seta_direita" = ir para direita;

    "a" ou "seta_esquerda" = ir para esquerda;

    "p" = pausar o jogo.
